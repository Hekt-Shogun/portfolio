# Deva Mithran — Portfolio

Portfolio site for Deva Mithran, a game artist and character designer based in Chennai, India.

"Every character starts as a shape before it becomes a face." The work here follows a silhouette-first design philosophy — block the shape, find the read, then chase detail only as far as it serves the character.

## What's on it

- **About** — Deva's background: a B.Sc. graduate in Game Design & Development, currently building a character art portfolio one sketch at a time and job hunting for game artist / character artist roles.
- **Skills** — character design, digital illustration, anatomy & figure drawing, concept sketching, color & lighting, inking, and both fan art and original character design.
- **Projects** — six character pieces spanning original designs and fan studies: an armored mercenary (Winter Reaver), a Ghost Rider anatomy study (Spirit of Vengeance), character portraits (Old Guard, Violet Hollow), a caped superhero design (Scarlet Vow), and a raw process sketch (Quiet Stare).
- **Timeline** — from graduating with a B.Sc. in Game Design & Development, through ongoing fan art and original character studies, to the current search for a game artist role.
